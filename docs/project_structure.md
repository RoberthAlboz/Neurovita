# Estrutura de Pastas do Projeto Neurovita

```text
Neurovita/
├── api/                # Backend (PHP API)
│   ├── admin/          # Gestão Administrativa
│   ├── ai/             # Integração com IA
│   ├── auth/           # Autenticação
│   ├── medical/        # Prontuários Eletrônicos
│   └── setup_db.php    # Inicialização do Banco
├── assets/             # Recursos de Documentação
├── docs/               # Documentos e Orientações
├── public/             # Arquivos Estáticos e 3D
├── src/                # Frontend (React + TS)
│   ├── components/     # Componentes de UI
│   ├── pages/          # Páginas da Aplicação
│   └── styles/         # Estilização Modular
├── start_project.sh    # Script de Inicialização
└── vite.config.ts      # Configuração do Ambiente
```
